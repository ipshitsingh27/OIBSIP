# Customer Segmentation Analysis

## Oasis Infobyte Internship

### Task 2 – Customer Segmentation Analysis

## Objective

The objective of this project is to segment customers based on their purchasing behaviour using K-Means clustering and RFM (Recency, Frequency, Monetary) analysis. The identified customer segments help businesses design targeted marketing strategies.

## Dataset

**Dataset:** Online Retail Dataset

The dataset contains customer transactions including invoice details, quantity purchased, unit price, invoice date, and customer ID.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow

- Loaded and inspected the dataset
- Cleaned missing and inconsistent data
- Calculated descriptive statistics
- Created RFM features
- Standardized the data using StandardScaler
- Applied the Elbow Method to determine the optimal number of clusters
- Performed K-Means clustering
- Visualized customer clusters
- Profiled each customer segment
- Recommended marketing strategies
- Exported customer segments
