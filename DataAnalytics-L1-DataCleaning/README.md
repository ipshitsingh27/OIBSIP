# Data Cleaning Project

## Oasis Infobyte Internship

### Task 3 – Data Cleaning

### Objective
The objective of this project is to clean a raw dataset by identifying and handling missing values, duplicate records, inconsistent formatting, incorrect data types, and outliers. The cleaned dataset is saved as a new CSV file for further analysis.

## Technologies Used

- Python
- Pandas
- NumPy
- Jupyter Notebook

## Project Steps

- Loaded the dataset
- Generated a data quality report
- Handled missing values
- Removed duplicate records
- Standardized categorical values
- Detected outliers using the IQR method
- Corrected data types
- Compared the dataset before and after cleaning
- Exported the cleaned dataset as `cleaned_dataset.csv`

## Files

- `Ipshit_Task3_Data_Cleaning.ipynb`
- `dirty_dataset.csv`
- `cleaned_dataset.csv`
- `README.md`

## Outcome

The dataset was successfully cleaned and transformed into an analysis-ready dataset suitable for further exploratory data analysis and machine learning tasks.?
